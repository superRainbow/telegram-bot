# telegram-bot

----

## 工具
使用以下工具

```
mongoose (MongoDB)
exceljs (產報表工具)
telegraf (telegram 套件)
```

---

## 使用

安裝相依性套件
```
npm i
```

本地端建立 .env
```
TELEGRAM_BOT_TOKEN=機器人ID
MONGODB_URL=mongodb位置
```

---

## 執行方式

叫起機器人
```
npm run build
```
